# 使用说明

## 1. 运行环境
- Windows 10 / 11
- JDK 21 或以上
- Maven 3.9 或以上（源码运行时需要）

## 2. 源码运行方式
在 `code` 目录执行：

```powershell
mvn javafx:run
```

如果本机 Maven 默认仓库目录没有写权限，可以使用：

```powershell
mvn "-Dmaven.repo.local=.m2" javafx:run
```

## 3. 打包方式
在 `code` 目录执行：

```powershell
mvn clean package
```

如果遇到 Maven 仓库权限问题，可以使用：

```powershell
mvn "-Dmaven.repo.local=.m2" clean package
```

打包完成后，会在 `target` 目录生成：

- `english-chinese-dictionary-1.0.0.jar`
- `lib/` 运行依赖目录

将这个 jar 和 `lib` 目录复制到 `bin` 后，可通过 `run.bat` 启动。

## 4. 使用方法
- 在搜索框输入英文单词，例如 `apple`
- 或输入中文关键词，例如 `学习`、`挑战`
- 按回车或点击“立即查询”
- 点击联想列表可快速补全查询
- 点击“收藏当前词条”可加入收藏
- 点击左侧历史或右侧收藏可重新查看词条

## 5. 数据存储说明
- 程序会在当前运行目录下创建 `app-data`
- `history.json` 保存最近查询
- `favorites.json` 保存收藏词条

## 6. 常见问题
### 6.1 软件无法启动
- 请确认已安装 JDK 21 或以上
- 如果使用源码运行，请确认 Maven 可用

### 6.2 查询不到内容
- 当前版本为本地演示词库，不是完整商用词典
- 可尝试输入词条中的关键英文前缀或中文核心词

### 6.3 收藏未保存
- 请检查当前运行目录是否有写入权限
